# POPSpring

> 调试POP的Spring动画效果

![demo.gif](http://images.cnitblog.com/blog2015/607542/201505/141455378768458.gif)


* POP动画中的Spring弹簧动画因为没办法计算出动画所需要的时间,所以,本人写了一个demo用以测试不同参数下的弹簧动画效果以及花费的时间.

* 本人写了一个小控件RangeValueView,可以单独拿出来用,很方便.
